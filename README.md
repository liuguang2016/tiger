# A 股盈利交易技术分析系统

分析您的 A 股交割单，自动筛选盈利交易，展示 K 线图 + 均线 + 成交量 + 买卖点标记，帮助您发现自己的盈利交易模式。

## 功能特性

- 上传 CSV 格式交割单，自动解析常见券商格式
- FIFO 方式配对买卖交易，计算每笔盈亏
- 筛选盈利交易，展示详细列表
- 交互式 K 线图（ECharts）：蜡烛图、MA5/10/20/60 均线、成交量
- 买卖点在 K 线图上标记（红色买入箭头、绿色卖出箭头）
- 交易模式统计面板（盈利总额、平均持仓天数、月度分布等）

## 安装与运行

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动应用
python app.py

# 3. 打开浏览器访问
# http://127.0.0.1:5000
```

## 交割单 CSV 格式

系统支持常见券商导出的交割单 CSV，典型字段包括：

| 字段 | 说明 |
|------|------|
| 成交日期 | 交易发生日期 |
| 证券代码 | 股票代码（如 000001） |
| 证券名称 | 股票名称 |
| 买卖标志 | 买入 / 卖出 |
| 成交价格 | 成交单价 |
| 成交数量 | 成交股数 |
| 成交金额 | 总金额 |
| 手续费 | 佣金 |
| 印花税 | 印花税 |
| 过户费 | 过户费 |

如果您的交割单列名不同，系统会尝试自动匹配。

## 技术栈

- 后端：Python + Flask
- 数据处理：pandas
- 行情数据：akshare
- 前端图表：ECharts
