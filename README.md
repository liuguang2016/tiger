# Tigger - A 股交易分析系统

集交割单分析与弹簧反弹选股于一体的 A 股交易辅助工具。分析历史交割单发现盈利模式，同时基于价格行为学筛选大幅下跌后企稳放量的反弹机会。

## 功能特性

### 交割单分析

- 上传 CSV 格式交割单，自动解析常见券商格式
- FIFO 方式配对买卖交易，计算每笔盈亏
- 盈利/亏损交易分类展示，支持多维度排序
- 交互式 K 线图（ECharts）：蜡烛图、MA5/10/20/60 均线、成交量、买卖点标记
- 交易统计面板：总盈亏、胜率、平均持仓天数等
- 交易风格分析报告：持仓天数分布、月度盈亏、板块偏好、个股 TOP10 等

### 弹簧反弹选股

基于价格行为学理论，筛选大幅下跌后"跌不下去"且成交量异变的股票：

- **全 A 股扫描**：一键筛选 5000+ 只 A 股（自动排除 ST、停牌、新股）
- **三重条件过滤**：
  - 跌幅达标 — 当前价距 60 日高点回撤 ≥ 阈值（默认 25%）
  - 企稳信号 — 最近 5 日不再创新低
  - 量能异变 — 近 3 日有单日成交量 > 20 日均量 2 倍
- **大盘参考**：结合上证指数当日走势评估信号有效性
- **综合评分排序**：跌幅深度、量比强度、振幅收窄、逆势表现多维度打分
- **交易池管理**：筛选结果持久化到数据库，支持逐只移除或清空
- **K 线图查看**：点击交易池中的股票即可查看近期 K 线走势
- **可调参数**：跌幅阈值（20%/25%/30%/35%）、量比阈值（1.0/1.5/2.0/3.0）

## 安装与运行

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动应用
python app.py

# 3. 打开浏览器访问
# http://127.0.0.1:5000
```

## 交割单 CSV 格式

系统支持常见券商导出的交割单 CSV，典型字段包括：

| 字段 | 说明 |
|------|------|
| 成交日期 | 交易发生日期 |
| 证券代码 | 股票代码（如 000001） |
| 证券名称 | 股票名称 |
| 买卖标志 | 买入 / 卖出 |
| 成交价格 | 成交单价 |
| 成交数量 | 成交股数 |
| 成交金额 | 总金额 |
| 手续费 | 佣金 |
| 印花税 | 印花税 |
| 过户费 | 过户费 |

如果您的交割单列名不同，系统会尝试自动匹配。

## 项目结构

```
tigger/
├── app.py                  # Flask 主应用，API 路由
├── requirements.txt        # Python 依赖
├── services/
│   ├── parser.py           # 交割单 CSV 解析
│   ├── matcher.py          # 买卖配对与盈亏计算
│   ├── analyzer.py         # 交易风格分析
│   ├── stock_data.py       # K 线数据获取（腾讯/东方财富/akshare）
│   ├── database.py         # SQLite 数据存储
│   └── screener.py         # 弹簧反弹选股引擎
└── static/
    ├── index.html          # 前端页面
    ├── css/style.css       # 样式
    └── js/
        ├── app.js          # 前端交互逻辑
        └── chart.js        # K 线图渲染
```

## 技术栈

- 后端：Python + Flask
- 数据处理：pandas
- 行情数据：东方财富 API / 腾讯财经 API / akshare
- 前端图表：ECharts
- 数据存储：SQLite
