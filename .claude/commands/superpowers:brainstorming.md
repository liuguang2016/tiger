# Superpowers Brainstorming

I'm using the **superpowers:brainstorming** skill to help shape your ideas into fully formed designs.

## Overview

This skill follows a structured brainstorming process:

1. **Explore project context** — understand current state
2. **Ask clarifying questions** — one at a time, understand purpose/constraints/success criteria
3. **Propose 2-3 approaches** — with trade-offs and recommendations
4. **Present design sections** — get approval before moving on
5. **Write design doc** — save to `docs/superpowers/specs/`
6. **Transition to implementation** — invoke writing-plans skill

## Start

What would you like to build or improve? Let me guide you through understanding what you're really trying to accomplish before we write any code.
