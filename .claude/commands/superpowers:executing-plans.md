# Superpowers Executing Plans

I'm using the **superpowers:executing-plans** skill.

## Prerequisites

This skill requires a valid implementation plan file. Common locations:
- `docs/superpowers/plans/*.md`

## Usage

To use this skill, please provide:
1. The path to your implementation plan, OR
2. A description of what you want to implement

Then I will:
1. Load and review the plan
2. Execute tasks step by step
3. Present checkpoints for review
4. Report when complete
