# Superpowers Verification Before Completion

I'm using the **superpowers:verification-before-completion** skill.

## Rule

**Evidence before assertions.** Never claim work is complete without running verification.

## Usage

Before declaring success, you MUST:
1. Run verification commands
2. Confirm output matches expectations
3. Document what was verified
