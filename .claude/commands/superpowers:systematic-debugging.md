# Superpowers Systematic Debugging

I'm using the **superpowers:systematic-debugging** skill.

## Overview

A 4-phase approach to debugging:
1. **Observe** — gather symptoms, error messages, context
2. **Hypothesize** — form testable theories about root cause
3. **Investigate** — design and run experiments to test hypotheses
4. **Verify** — confirm fix works and doesn't break other things

Use this skill when encountering any bug, test failure, or unexpected behavior.
