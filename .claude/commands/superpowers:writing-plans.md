# Superpowers Writing Plans

I'm using the **superpowers:writing-plans** skill.

## Overview

Create comprehensive implementation plans that provide everything a developer needs:
- Exact file paths for each task
- Complete code in the plan
- Exact commands with expected output
- DRY, YAGNI, TDD principles
- Frequent commits

## Output

Plans are saved to: `docs/superpowers/plans/YYYY-MM-DD-<feature-name>.md`
